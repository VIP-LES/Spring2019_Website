Mission Objectives

____________

  • Flight Computer

    Testing the new, lighter flight computer and the accompanying APRS transmitter.

  • Platform

    Testing the new platform design based only on a single Raspberry Pi and Python code.

  • Geiger Counter

    Testing the new Geiger Counter sensor, its accuracy and its communications.

  • Inertial Measurements

    Testing the new compass/gyro/accelerometer inertial measurement unit.
